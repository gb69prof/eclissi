# ECLISSI — PWA v3 precisa

Versione rifatta con:

- coordinate iniziali di Foggia: **41°27′42″N, 15°32′42″E** = 41.461667, 15.545000;
- pannello spiegazioni fuori dalla scena, quindi la simulazione non viene coperta;
- distinzione reale nella vista al suolo tra eclissi totale, anulare, parziale e non visibile;
- calcolo locale tramite Astronomy Engine quando il browser riesce a caricare la libreria online;
- fallback didattico se la libreria non si carica.

Fonti principali: NASA/GSFC Eclipse Web Site, Fred Espenak; Astronomy Engine 2.1.19, Don Cross.

Nota di onestà: la PWA usa calcolo locale serio quando Astronomy Engine è attiva. La resa grafica resta una simulazione visiva didattica, non uno strumento certificato da osservatorio.
